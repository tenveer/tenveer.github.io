  • Snippet: To check if App is installed via TestFlight

    Sunday, January 26, 2020

    Apple provides a way to validate receipts, related to that the follwing code snippet can be used to verify app is installed from TestFlight or not. func isInstalledFromTestFlight() -> Bool { guard let appStoreReceiptURL = Bundle.main.appStoreReceiptURL else { return false } return appStoreReceiptURL.lastPathComponent == "sandboxReceipt" } Related Validating Receipts with the App Store
